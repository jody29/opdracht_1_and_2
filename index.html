Browser compatibility

Browser compatibility is de mogelijkheid of flexibiliteit van een website, web applicatie, script of HTML design om op verschillende web browsers nog steeds te functioneren.

Niet alle functionaliteiten, html-elementen of css-selectoren worden door elke browser ondersteund. Daarom is het van belang om jezelf af te gaan vragen of een bepaald iets wat je wilt gebruiken wel door veel browsers ondersteund wordt. Voor het zelfde geldt gebruik je een webAPI die alleen in Chrome werkt of gebruik je een css-selector die in heel weinig browsers wordt ondersteund.

Om te checken of een browser een functionaliteit ondersteund kun je gebruik maken van https://caniuse.com/
